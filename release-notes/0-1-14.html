Private Bookmarks 0.1.14

These are the release notes for Private Bookmarks 0.1.14, released on 13/01/2018.

What's new
  • More characters allowed in passwords: You can now use any of the printable characters in the ASCII set.
  • Prefer a dark theme? You can now switch between light/dark themes for the main panel (it's the brush-shaped button at the bottom).

Something's not working? Got a question and/or suggestion? Post it here.

This is a reminder that the extension is quite new and that things may break, so please backup your bookmarks periodically to be safe.

If you do not wish to be notified of new features after an update, you can un-subscribe through the extension's options page.
