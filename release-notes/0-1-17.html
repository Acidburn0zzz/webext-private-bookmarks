Private Bookmarks 0.1.17

These are the release notes for Private Bookmarks 0.1.17, released on 16/09/2018.

What's new
  • Custom folder name: Just like you can customize the folder's location, you can now customize its name. Once unlocked, if you rename it Private Bookmarks will remember the new name.
  • New option for periodic backup reminders. It is enabled by default.
  • Starting with Firefox 63 shortcuts with Alt+Ctrl are supported.
  • Bookmark all tabs in a window at once using a new shortcut. By default it is Ctrl+Shift+F8.

Something's not working? Got a question and/or suggestion? Post it here.

As always with a new release, be on the lookout for bugs and oversights, and make backups frequently!

If you do not wish to be notified of new features after an update, you can un-subscribe through the extension's options page.
