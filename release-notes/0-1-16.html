Private Bookmarks 0.1.16

These are the release notes for Private Bookmarks 0.1.16, released on 22/02/2018.

What's new
  • Starting with Firefox 60 (current Nightly) it is possible for extensions to update their command keyboard shortcuts. If you are a Nightly user, you should now have access to a new 'Shortcuts' section in the options page.

Something's not working? Got a question and/or suggestion? Post it here.

This is a reminder that the extension is quite new and that things may break, so please backup your bookmarks periodically to be safe.

If you do not wish to be notified of new features after an update, you can un-subscribe through the extension's options page.
