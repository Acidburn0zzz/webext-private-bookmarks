Private Bookmarks 0.1.15

These are the release notes for Private Bookmarks 0.1.15, released on 29/01/2018. This release contains one new feature, some bug fixes and small performance improvements.

What's new
  • Accidentally left your bookmarks unlocked? You can now configure Private Bookmarks to auto-lock when the system's been idle for a specified number of minutes. Look for a new checkbox under the 'General' section of the options page.

Something's not working? Got a question and/or suggestion? Post it here.

This is a reminder that the extension is quite new and that things may break, so please backup your bookmarks periodically to be safe.

If you do not wish to be notified of new features after an update, you can un-subscribe through the extension's options page.
