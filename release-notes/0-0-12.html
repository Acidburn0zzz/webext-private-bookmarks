Release Notes

A new version of Private Bookmarks has been installed.

What's new
  • Additional characters are now allowed in passwords: ~!@#$%^&*_-+= and space.
  • Private Bookmarks remembers where you move it to: If you move your unlocked private bookmarks folder somewhere, the next time you unlock it will appear there instead of under 'Other Bookmarks'.
  • Not a fan of our password requirements? You can disable them: The default requirement for passwords to be at least 8 characters long and contain at least one letter and one number helps making your passwords stronger. But, if you don't care about that you can now disable these requirements from the options page.
  • Import backup from several files at once: You can now select multiple files to import from (provided they share the same password).

Something's not working? Got a question and/or suggestion? Post it here.

This is a reminder that the extension is very new and that things may break, so please backup your bookmarks.

If you do not wish to be notified of new features proceeding an update in the future, you can un-subscribe through the extension's options page.
